# Project Brief

## Objective
Build a Windows 11 WPF application that broadcasts a narrow allowlist of keyboard inputs from one leader window to one follower window.

## Default Allowed Keys
- 1 2 3 4 5 6 7 8 9 0 - =

## Exclusions by Design
- all other keys
- all modifier-assisted input
- mouse movement
- mouse buttons
- mouse wheel

## v0.1 Success Definition
- one leader selected manually
- one follower selected manually
- allowlisted keys broadcast reliably
- non-allowlisted keys ignored
- modifiers ignored
- logs available for diagnostics
